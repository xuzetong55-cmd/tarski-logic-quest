# Private Content

This directory is for local study material that should not be committed.

Use it for:

- Personal LPL exercise notes
- Your proof attempts
- Your custom Tarski worlds
- Local answer drafts
- References to textbook exercises you own

Do not put copyrighted exercise text into the public repository. The repository tracks this README and `lpl/.gitkeep` only; everything else under `private-content/` is ignored by git.

Suggested layout:

```text
private-content/
  lpl/
    ch01/
      1.1.md
      1.2.md
    ch02/
      2.1.md
```

Suggested note template:

```md
# LPL Exercise X.Y

## Reference

Short private note about where the exercise appears.

## Attempt

Your solution attempt.

## Feedback

What to revisit.
```
